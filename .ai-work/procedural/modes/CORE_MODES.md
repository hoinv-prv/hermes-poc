# Core Modes

## BrainStorming
Dùng để mở ý tưởng, so sánh option, thử simulation, phát hiện missing points.

## Research
Dùng để điều tra, đọc tri thức, làm rõ dependency, thu thập evidence.

## Planning
Dùng để tạo approach, scope, step skeleton, handoff package.

## Executing
Dùng để thực hiện output thật, cập nhật runtime state, tạo deliverable.

## Reviewing
Dùng để self-check / review quality / detect issues.

## Organizing
Dùng để整理 findings, compact queue, structure notes, prepare curated updates.
