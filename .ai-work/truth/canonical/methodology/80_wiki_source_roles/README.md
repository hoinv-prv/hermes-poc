# Wiki Source Role Package for AI Work System MVP
Version: 0.1
## Terminology note
Trong AI Work System:
- **Knowledge Hub** là tên chính thức của component
- **Wiki** là tên vắn tắt vẫn được dùng bên trong hệ thống để chỉ Knowledge Hub


## Purpose
Package này chốt cách đưa **source working** và **source legacy** vào Knowledge Hub / Wiki system của AI Work System MVP.

## Included
- `WIKI_SOURCE_ROLE_POLICY_MVP.md`
- `AI_WORK_SYSTEM_UPDATE_GUIDE_FOR_SOURCE_ROLES_MVP.md`
- `GUIDE_ADD_WORKING_AND_LEGACY_SOURCES_TO_PROJECT_WIKI_MVP.md`

## Intended use
Dùng package này để:
- chốt policy cho working vs legacy source
- update nhẹ design/spec/package hiện tại của AI Work System
- hướng dẫn đưa 2 bộ source vào wiki của một dự án phần mềm
