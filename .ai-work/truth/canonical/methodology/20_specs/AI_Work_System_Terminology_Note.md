# AI Work System Terminology Note
Version: 0.1  
Status: Canonical terminology note

## Official term
Within AI Work System, the official component name is:

- **Knowledge Hub**

## Shorthand
Inside the system, the shorthand term:

- **Wiki**

is still intentionally used as the short name for **Knowledge Hub**.

## Interpretation rule
When a document in this package says:
- Wiki
- Project Wiki
- Local Wiki
- Wiki Source
- Wiki update
- Wiki lookup

it should be interpreted as referring to:
- the **Knowledge Hub** layer or its related source-side / curated-side structures,
unless the surrounding context clearly points to a narrower technical folder/path convention.

## Why this note exists
The design has evolved from earlier “Wiki-first” language into a broader **Knowledge Hub** concept.
However, the short word **Wiki** remains useful and is intentionally retained as a lightweight operational name inside AI Work System.
