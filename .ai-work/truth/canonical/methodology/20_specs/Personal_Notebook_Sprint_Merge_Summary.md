## 3. Canonical decisions

1. Personal Notebook is a file/folder-based personal working reference area.
2. Personal Notebook is separate from Workspace findings, Working AIP, and Knowledge Hub.
3. Personal Notebook effective scope follows configured notebook path.
4. Personal Notebook is not source of truth by default.
5. Personal Notebook does not auto-promote content into Knowledge Hub.
6. Personal Notebook can create capture candidates.
7. Personal Notebook Write Skill Lite is allowed as a lightweight, customizable support capability.
8. The skill is not decision authority, orchestrator, or promotion pipeline.

- Task Notebook / Workspace Notebook

- notebook UI/database
- multi-user notebook
- full governance/access control
- notebook scoring/telemetry
- full capture request automation
