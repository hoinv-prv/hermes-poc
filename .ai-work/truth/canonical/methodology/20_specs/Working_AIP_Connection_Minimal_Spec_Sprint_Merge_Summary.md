## 3. Key decisions

1. Discovery/reuse/context can inform execution.
2. Working AIP controls execution.
3. Before non-trivial execution, AI must have or create a Working AIP.
4. Support artifacts can feed Working AIP but cannot replace it.
5. run-aip executes against Working AIP.
6. If Working AIP is missing or not ready, prepare/update it first.
7. Candidate collection is not promotion or apply-back.
