# wiki_candidates/ — reviewable Wiki candidates / patches (PRE-approval)

Holds Wiki lifecycle artifacts produced by Phase G+ agents. Everything here is a **candidate for
HUMAN review** — clearly separate from Official Wiki, never auto-applied.

- `candidate_packs/` — Wiki Candidate Packs (proposed pages + index patch + source trace + knowledge-value check)
- `patch_proposals/` — Wiki Patch Proposals for changed source/current Wiki (+ stale/conflict reports)
- `review_packs/` — Wiki Build Review Packs (HUMAN review checklist + approval recommendation)

Empty until Phase G. Official Wiki update requires explicit HUMAN approval (no silent promotion).
