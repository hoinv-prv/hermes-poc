# wiki_meta/ — GENERATED metadata (NOT Official Wiki)

Holds metadata produced by metadata sub-agents (Phase E+). This is **generated metadata for
navigation/understanding/support**, never Official Wiki content and never auto-promoted.

- `inventories/` — folder/document/code/DB/API inventories (`*.jsonl` + source trace)
- `relations/` — relation indexes across docs/source/API/DB/modules
- `meta_reviews/` — Meta Quality Review outputs (coverage/conflict/gap reports)
- `tool_outputs/` — raw outputs of metadata extraction tools

Empty until Phase E. Separation from `wiki_candidates/` and Official Wiki is a guardrail.
